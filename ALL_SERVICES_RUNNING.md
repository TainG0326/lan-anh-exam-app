# 🎉 Tất cả services đã được khởi động!

## ✅ Trạng thái:

- ✅ **Backend Server**: http://localhost:5000
- ✅ **Teacher Web**: http://localhost:3001 (đang khởi động)
- ✅ **Student Web**: http://localhost:3002 (đang khởi động)

## 🔗 Truy cập:

### Teacher Portal:
- **URL**: http://localhost:3001
- **Tài khoản**: teacher@example.com
- **Mật khẩu**: teacher123

### Student Portal:
- **URL**: http://localhost:3002
- **Tài khoản**: student@example.com
- **Mật khẩu**: student123

### API Server:
- **URL**: http://localhost:5000
- **Health Check**: http://localhost:5000/api/health

## ⏳ Đợi vài giây...

Các web frontend cần vài giây để compile và khởi động. Sau đó:

1. Mở browser
2. Truy cập http://localhost:3001 (Teacher) hoặc http://localhost:3002 (Student)
3. Đăng nhập với tài khoản mặc định

## 📝 Lưu ý:

- Nếu web chưa load, đợi thêm 10-15 giây
- Kiểm tra terminal để xem có lỗi không
- Đảm bảo backend server (port 5000) đang chạy

## 🚀 Bắt đầu sử dụng:

1. **Teacher**: Đăng nhập → Tạo lớp → Tạo kỳ thi → Xem kết quả
2. **Student**: Đăng nhập → Xem kỳ thi → Nhập mã thi → Làm bài

Chúc bạn sử dụng tốt! 🎓
